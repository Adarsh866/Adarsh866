### 💼 What I've Built

#### 🎓 Online Classes Streaming Platform
Live class streaming platform serving **4 user roles** — students, teachers, admins & super admins — with CDN-delivered course resources.
Microservices behind an API gateway (REST for clients, **gRPC + Kafka** between services), JWT auth + RBAC, Redis caching that cut response times, real-time classrooms over WebSockets, and Docker + GitHub Actions CI/CD.

`React.js` `NestJS` `MongoDB` `Redis` `Kafka` `gRPC` `Docker`

#### 🏪 Multi-Shop SaaS Management Platform
Multi-tenant SaaS onboarding retail shops onto a centralized database, with dedicated **admin** and **vendor** panels.
CRUD REST APIs with NestJS + TypeORM secured by JWT & role-based access, microservices + API gateway with Redis caching for fast multi-tenant reads, containerized and shipped via CI/CD.

`React.js` `NestJS` `MySQL` `TypeORM` `Redis` `Docker`

#### 🚚 Real-Time Logistics Delivery App <sub>*(side project)*</sub>
Delivery app with role-based access for **riders, customers & admins** — WebSockets for live order/delivery status updates and Redis for low-latency reads. Built fast with AI-assisted development.

`React.js` `NestJS` `TypeScript` `WebSockets` `Redis`
